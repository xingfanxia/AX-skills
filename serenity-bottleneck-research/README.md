# Serenity Bottleneck Research Skill

This bundle contains a research-process skill inspired by Serenity-style bottleneck hunting. It is designed for agentic research workflows and should produce structured diligence memos, not trading recommendations.

Files:

- `SKILL.md`: main skill instructions.
- `templates/thesis_card.md`: reusable thesis card.
- `templates/evidence_ladder.md`: source-quality rubric.
- `examples/example_output_skeleton.md`: final memo skeleton.

Install by placing the folder in your agent/skill environment and invoking it for industry, supply-chain, technology-theme, or company-thesis analysis.
