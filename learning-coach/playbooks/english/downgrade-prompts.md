# 降维 prompt 模板(给学习者自助用)

> 动作 A 的降维默认由当前 agent 在会话内完成。本文件是一份**可复制给学习者的 prompt**,让她在自己手机上的 AI(豆包/DeepSeek/任何她有的)里自助把某段英文降维——用于她自己想再看某段、或 AX 不在身边时。
> AX 先帮她把常用那条存进微信收藏,她用时复制即可——不用自己打。

## 主力:整段 i-1 降维(三栏)

```
把下面这段英文,用最简单的英语重写一遍(只用最常见的词、句子更短、不要用新的难词、小学生能看懂),
保留原意和关键信息,不要丢内容。然后:
1) 给出完整中文翻译;
2) 挑 3-5 个有用的词,每个给中文意思 + 一句话怎么用。
不要让我背任何词,只标出来就行。

英文原文:
"""
<把英文粘到这里>
"""
```

## 轻量:划词/单句解释(她看视频/读东西时随手用)

```
这句英文什么意思?用最简单的中文讲一下,再告诉我里头哪个词最有用、怎么用:
"<把那句话粘到这里>"
```

## 学 AI 闭环:边学概念边吸英文词

```
我想搞懂 <某个 AI 概念,如 AI agent>,用最简单的中文给我讲清楚,
再教我这领域最常见的 3 个英文词,每个给中文意思 + 一句话怎么用(我不背,认脸就行)。
```

## AI 安全核查(养成肌肉记忆)

```
What is your source?    /    这个信息你的来源是什么?
Is this recent?         /    这是最新的吗?
How can I verify this?  /    我怎么核实这个?
```

## 给 AX 的提醒

- 这些 prompt 的产物**仍要符合**三栏精神:简化版仍是英文(不是只给中文)、词只标不要求背。
- 这些 prompt 是给学习者自助用的;skill 在 AX 端的降维直接用 agent 自身的模型,不经过这里。
- 别一次给她甩一堆 prompt——按场景给一个,降低她的选择负担。
