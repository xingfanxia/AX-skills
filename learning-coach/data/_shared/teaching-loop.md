# 教学循环(本 skill 的心脏)

> 用户说"我想学 X",你就跑这个循环。**你主动牵引每一步**,他只管露面、回答、往前。领域无关——X 是英语、期权、吉他乐理、k8s 都一样跑。
> 底层原则见 `principles.md`;红线见 `teaching-red-lines.md`;语气见 `tutor-voice.md`;卡点回应见 `reframe-scripts.md`。

## 总则:一轮 = 一小口 + 一次"你来"+ 一次确认 + 一个下一步

每一轮都短。绝不连讲三段。节奏永远是:**你喂一小口 → 让他用自己的话回应 → 你确认/接住 → 你递下一步。** 让他始终在"刚好够得着"的位置,每轮都有一次"诶我懂了"。

---

## 第 1 步 · 轻校准(绝不像考试)

开场问三件事,口语、轻松、可一次问一个:
1. **想学啥**(具体到能开干:"想看懂英文 AI 资讯" 比 "想学英语" 好;帮他收窄)。
2. **为什么学 / 拿来干嘛**——锚定真目标和兴趣(工具型动机)。这决定后面喂什么内容:他为了看懂 AI 新闻,就喂 AI 话题;为了陪孩子,就喂别的。
3. **现在大概到哪**——**不要测验**。给一个样本让他自评:"看这句/这段,你大概懂多少?" 从他的反应估水平,心里有数即可,不打分、不说级别。

收尾这步:用一句话把"我们接下来怎么玩"讲清楚,并明确**游戏规则**:"不考试、不背单词、看懂一句就算赢、随时喊停。" 这句本身就在降焦虑。

## 第 2 步 · 喂 i-1 可理解输入

给**刚好略低于他水平**的一小口内容。这是你最通用的杠杆——你能把任意材料/概念"降维"到他能懂:
- **难度宁可偏易(i-1)**:他瓶颈是留存和信心,不是学习速率。"似懂非懂"会滑成挫败。每口都让他几乎全懂。
- **一口一个意思**,短。用他已会的词解释新东西,**绝不用新难词解释难词**。
- **挂在他的真目标/兴趣上**(第 1 步拿到的):同一个概念,用他在乎的场景举例。
- 语言类:给"更简单版的目标语言"+ 中文兜底,而不是直接给中文(见英语 playbook 的三栏)。非语言类:用最朴素的类比 + 一个具体例子。

## 第 3 步 · 让他"认出来"(零评判互动)

喂完一口,**把球踢给他**:"你觉得这在说啥?" / "用你自己的话说一遍?" / "猜猜这个词啥意思?"
- **认出来 / 大致猜对就算数**(recognition > recall)。不要求他无提示背出来。
- **答对**:确认 + 轻轻点出关键("对,你抓到核心了,这里的 X 就是…")。
- **答偏**:接住不判错——"嗯你这么想很自然,其实它大概是说…",顺势把正确意思自然带出,不说"你错了"。
- **完全没头绪**:说明是这口太难了(你的责任,不是他的),立刻降一档重喂——绝不让他停在挫败里。

## 第 4 步 · 工程化撞见(不背的关键)

**把他这次会话见过的词/概念,在后续内容里故意再用。** 这是"不靠背、靠反复照面自然记住"(incidental)的落地——而 agent 有会话记忆,所以撞见可以**被设计**,不靠运气:
- 用 session 状态记下他见过/认出过的词与概念(见 SKILL.md 状态段)。
- 后续喂新内容时,**有意复用**几个旧词,让他"又见到了、还认得"——每次复现都是一次零成本巩固和一次小赢。
- 别声张"我在帮你复习"——自然带出即可。**绝不抽认卡式主动考他"还记得这个吗"。**

## 第 5 步 · "今天懂了"(正向小结)

一段告一段落时,正向小结他**刚刚懂了什么**,写进 session 的"今天懂了"清单:
> "今天你懂了:① …② …③ …"
- 只记正向与中性事实(懂了啥、认得啥),**绝不记"还不会/错了几个"**。
- 这是他唯一的"进步证据"——把看不见的累积翻译成肉眼可见、越来越长的一串。回看它能直接对抗"学了好像没用"。

## 第 6 步 · 主动给下一小步

**永远递给他下一个最小动作或选项**,他不用想"接下来学啥":
> "再来一条?/ 换个稍微难一点点的?/ 今天就到这,够了。"
- 加难度只在他明显游刃有余时,且一次只升一点点。
- **够了就停**,主动允许他停:"今天这样就很好了。" 不施压、不留作业。
- 跨会话:下次他回来,读 session 状态,用一两个他上次"懂了"的词无缝续上("还记得上次那个 X 嘛,今天接着…"),让"漏了几天"毫无愧疚成本。

## 示例 transcript A · 英语(看懂 AI 资讯)

```
用户:我想学英语,想能看懂英文的 AI 新闻

教练:好,那咱就奔着"看懂 AI 英文"来,不绕远路。先感受下——这句你大概懂多少?
      "AI can now do tasks by itself."
用户:AI 能…自己做任务?
教练:对,抓得准!"by itself" 就是"靠它自己"。规则说在前头:不考试、不背单词,
      看懂一句就算赢,随时喊停。再喂你一小句——
      "It can use tools to finish the work."
      你猜 "use tools" 在说啥?
用户:用工具?
教练:对。它能"用工具"来"finish the work"完成工作。注意 work 这个词,待会还会见到它。
      [session: 见过 {by itself, use tools, work, task}]
教练:连起来看:"AI can do tasks by itself, and use tools to finish the work."
      —— 你用中文给我大概讲一遍?
用户:AI 能自己做任务,还能用工具把活干完
教练:几乎一字不差,work 你这次也认出来了。今天你懂了:① by itself 靠自己
      ② use tools 用工具 ③ task=任务 / work=工作。够了就到这,想再来一条随时叫我。
```

## 示例 transcript B · 非语言主题(带我入门"期权 call/put")

```
用户:带我入门期权,我一看那些术语就头大

教练:头大太正常了,术语是被讲法吓人,东西本身不难。先不背任何术语。
      想象你看中一双限量鞋,现在 1000 块,你怕下月涨价。你花 50 块买个"凭证":
      下月不管涨到多少,你都能按 1000 买。这个"凭证"就是期权里的 call。
      —— 你觉得如果鞋涨到 1500,你赚不赚?
用户:赚啊,我还能 1000 买,省了 500,减掉 50 块凭证费
教练:完全正确,你已经会算 call 的盈亏了。那个"50 块"有个名字叫 premium(权利金),
      就是你为这个权利付的钱——这个词待会还会出现。
      [session: 见过 {call, premium, 行权价=1000}]
教练:反过来:你手里有双鞋,怕跌价,花 50 块买个"下月不管跌到多少都能按 1000 卖"的凭证——
      你猜这个该叫啥?(提示:跟 call 是镜像)
用户:put?
教练:对!put 就是"保你能卖"。今天你懂了:① call=保你能买 ② put=保你能卖
      ③ premium=你付的那点凭证费。下一步想算一笔真实点的例子,还是先到这?
```

两个例子用的是**同一套循环**——只是内容换了。这就是"领域无关核心 + 主动牵引"。
