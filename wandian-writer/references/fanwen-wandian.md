# 晚点 LatePost 范文风格分析

写 ZH 深度长文之前，先读这份。这 6 篇是选定的 style anchor，规则训不出来的细节都在原文里。

> **关于示例**：本文中的具体公司名（Compute Labs / Mio / OpenClaw 等）和"AX"作者名均来自原始作者（[xingfanxia](https://github.com/xingfanxia)）的真实项目和博客，作为"建过东西的人在写"这种姿态的具体例子。读者请映射到自己的项目和身份——核心是"用真做过的事写"，不是套用具体名字。

## 范文目录

| # | 标题 | 作者 | 类型 | 字数 |
|---|------|------|------|------|
| W1 | 字节、阿里、腾讯 AI 大战全记录 | 高洪浩 / 宋玮编 | 行业战争 deep report | ~9000 |
| W2 | 详解 DeepSeek V4：Infra 巨鲸"四连击" | 程曼祺访谈 | 技术访谈整理 | ~10000 |
| W3 | 一代人有一代人的 VC | 祝颖丽 / 赵磊编 | 行业 trend 观察 | ~6000 |
| W4 | 永不停歇的效率工厂：字节造红果 | 郑可书 / 黄俊杰编 | 产业改造 deep report | ~7000 |
| W5 | 马斯克的 "2 万亿" 美元拼图 | 李赓 / 胡昊编 | 公司财务/战略拆解 | ~6500 |
| W6 | 全员 token-maxxing | 五源资本孟醒（专栏） | 一线投资人观察 | ~9000 |

W1-W5 = 记者深度报道，集体"我们"口吻。W6 = 专栏作者第一人称，**最接近 AX 博客该有的姿态**。

---

## 整体 thesis

> 晚点风格 = 极具体的开头 + 判断式小标题 + 短段+短句 + 数据先行 + 判断落段尾 + 类比有锚 + 不端一面之词。

不是 flowing prose（晚点几乎全部用 header）。不是激情口语（晚点冷静）。不是数据堆砌（晚点每段都有判断）。

---

## 12 个可学习模式

### 1. 开头：永远从一个具体场景/数字/人物动作切入

✅ W1 开头：
> 2025 年 11 月，姚顺雨穿着休闲短裤、踩着拖鞋出现在腾讯的一场内部会上。这位 27 岁的 OpenAI 前研究员、提出过 ReAct 范式的技术天才刚被腾讯以重金招入麾下不久。
>
> 入职后，他的一项重要任务是帮腾讯找到混元大模型长期表现欠佳的原因[...]
>
> "混元的评测出了大问题。" 一位在场人士转述姚顺雨在会上发言[...]

✅ W4 开头：
> 3 月，短剧制作公司老板张宽（化名）要处理这些问题：债务，裁员，组建数百人的 AI 漫剧团队，还有修复他因忙碌哑火的嗓子。

✅ W5 开头：
> 马斯克准备在 2026 年夏天从美股拿走 300 亿至 800 亿美元。

✅ W6 开头：
> 2026 年 3 月 24 日早上，我坐在 YC W26 batch Demo Day 的观众席里，听到第五家公司上台路演的时候，决定不再做笔记了。
>
> 不是不重要，而是我意识到，自己记下来的这些东西，可能下个月就过时了。

**值得偷的**：开头永远是**人 + 具体动作 + 时间 + 一句反常判断**。永远不是"在AI时代""随着…的发展"。最好的开头会让读者立刻问"为什么？"——为什么穿短裤拖鞋？为什么不做笔记了？为什么要拿走 800 亿？

---

### 2. 副标题（首段下方那行 italic）= 整篇 thesis 的最简表达

| 文章 | 副标题 |
|------|--------|
| W1 | 从算力到入口，巨头进入 "中途岛时刻" |
| W2 | 系统级耦合优化比单点创新更难 |
| W3 | 更小、更快、更独立，也更全球化 |
| W4 | 免费、自己爽、无限效率，三年统治所有环节 |
| W5 | SpaceX 递给市场的，是一个由猎鹰和星链现金流打底、星舰和 AI 推高估值、马斯克本人决定赔率的 "太空盲盒" |
| W6 | 我们去硅谷考察了一圈，发现连造浪的人，都快被浪淹没了 |

**值得偷的**：副标题不是"概述本文将分析X"，而是直接把 thesis 拍出来。3 种格式：
- 并列短句 (W3, W4)
- 单句判断 (W2)
- 复合句但只有一个动词中心 (W5, W6)

---

### 3. 小标题：判断式 + 动作式，从不描述性

✅ 优秀样本：
- W1: "DeepSeek 之后，战争真正开始了" / "豆包凶猛，一个超级入口的成形" / "元宝补课，从工程债到产品节奏" / "再造巴别塔，AI 会让世界更开放，还是更封闭？"
- W4: "靠效率赚钱，短剧不养闲人" / "社会情绪永远在变，短剧永远跟得上" / "番茄小说的成功路径，用红果再来一遍" / "被摧毁与被重建的"
- W6: "全员 token-maxxing：一场没人敢停的军备竞赛" / "xAI 团队雪崩之后，造火箭的人开始造模型了" / "焦虑的工程师，更焦虑的 researcher" / "酸橙树与 AI 暗杀名单"

❌ 晚点绝不会写的小标题：
- "市场分析"
- "AI 行业的现状与未来"
- "竞争格局"
- "用户行为研究"

**值得偷的**：小标题里要有**动作**（"补课"、"雪崩"、"造红果"）或**判断**（"战争真正开始了"、"短剧不养闲人"）或**反差**（"造火箭的人开始造模型了"）。一个 header 单独看就是一句完整的话/insight。

注意标点：晚点用冒号、引号、问号都很自由，**没有 mp 那种"全文禁冒号"**。这是流派差异。AX 跟晚点走，可以用冒号。

---

### 4. 段落极短，单句独立成段做停顿

晚点的段落普遍 2-4 行。**单句成段是常见的节奏工具**，不是稀有装饰。

✅ W1 样本（关注段落长度）：
> 三大互联网巨头中，阿里的主业赚钱效率最低。据报道，今年前三季度字节跳动的净利润在 400 亿美元左右；同期，腾讯与阿里的净利润为 300 亿美与 100 亿美元左右（自然年口径）。但这没有影响它投入的决心。
>
> 一位知情人士透露，阿里正考虑将未来三年投入到 AI 基建与云计算上的 3800 亿元提升至 4800 亿。

✅ W4 经典单句段：
> 那是 2012 年。

（前面铺了一整段景峰当年花 20 天给电影做孔雀尾翼的细节，然后突然——"那是 2012 年。"——做时间锚定，读者瞬间感受到时代差。）

✅ W6 单句段做反转：
> 这一次，得按 "周"。

**值得偷的**：信息密度高的段落 → 短段（2-3 句）。需要重锤的判断 / 反差 / 时代锚定 → 单句独立成段。每篇 5-10 个单句段，用在转折处。

---

### 5. 数据-观点 ≥ 3:1，每段都要有 anchor

✅ W5 火箭成本拆解段：
> 造一枚全新的猎鹰 9 号大约需要 5000 万美元。但经过 40 次的高频复用，单次发射的硬成本被极限压缩到了约 1500 万美元，这笔钱被掰成了三份：
> - 大头是那截无法回收、在大气层烧毁的二级火箭，约 750 万美元；
> - 把火箭运到发射台及操作费用，约 500 万美元；
> - 而装满火箭的航空煤油、液氧燃料和检修翻新费，其实只有约 150 万美元。

✅ W1 段：
> 据报道，今年前三季度字节跳动的净利润在 400 亿美元左右；同期，腾讯与阿里的净利润为 300 亿美与 100 亿美元左右（自然年口径）。但这没有影响它投入的决心。

**值得偷的**：观点之前先甩 2-4 个具体数字 / 公司名 / 人名 / 时间点。判断没有 anchor 就别下。"很多""大量""非常"零容忍。**这条是晚点和营销号最大的体感差异**。

---

### 6. 判断落点：放段尾，不放段首

✅ W4 经典：
> 即便作品表现平平，依然能通过压缩制作成本，从保底费里赚到差价。广告公司、婚庆公司、房地产行业的人都转行拍起了短剧。北京的咖啡馆重回 2016 年的热闹——人们激动地讨论短剧项目。**上一次还是十年前，讨论向爱奇艺、腾讯视频推销怎样的穿越剧。**

（前面铺事实，最后一句把读者拉回历史做对照，判断/insight 在尾。）

✅ W6 经典：
> 我没有得到正面回答。事实是，100 倍的效率提升，落到公司的营收增长上，只体现了 50% 或者 1 倍。
>
> 差距在哪？现在还没人能说清楚。
>
> **"用了这么多 token，公司应该基因突变成另外一种公司才对。但到底变成什么，我也不知道。"**

❌ 反例（AX 容易写成的形式）：
> 这次 AI 浪潮中，效率提升和营收增长之间出现了显著差距。具体而言：很多公司报告 100 倍的效率提升[...]

**值得偷的**：先讲事实/数据/案例，**最后一句**才是 insight。让读者自己感到"哦"，而不是上来就告诉他答案。这跟 mp/khazix 强调的"层层剥开的修辞"是同一件事，但晚点的版本更克制——只放一句就停。

---

### 7. 引用极频繁，且永远标"来源类型"，不标具体姓名

✅ W1 标准句式：
- "一位接近字节跳动高层的人士说"
- "一位混元大模型人士说"
- "据我们了解"
- "一位知情人士透露"
- "一位元宝人士告诉我们"

**值得偷的**（AX 写自己的内容时）：
- 用真实经历时直接说"我"（这是博客，不是匿名报道）
- 引用别人的判断/产品体验时，用"一位 XX 团队的朋友说""我们做 XX 的时候发现"
- **不要假装匿名**——AX 在 Compute Labs / Mio / OpenClaw，写到自己人就直说

晚点这套"匿名信源"做法是记者文体专属，AX 别套。但**引用密度可以学**——晚点的报道里几乎每段都有一句引用做 punctuation。

---

### 8. 转场词：极简，靠语义而非连接词

晚点的连接词观察（6 篇全文统计估算）：
- "但" / "不过" — 高频（每篇 20-50 次）
- "更..." — 高频做递进
- 时间推进词 "从此""到了 2025 年""一年后" — 高频
- "因为" / "所以" — 中频
- **"然而" — 0 次**（写报告才用）
- **"综上所述" — 0 次**
- **"值得注意的是" — 0 次**
- **"首先...其次...最后" — 0 次**
- **"本质上" — 极低频**
- **"换句话说" — 极低频**

**值得偷的**：mp 的禁用词表跟晚点风格是 100% 对齐的。但晚点比 mp 多了一个杀手锏——**直接换段不连**。两段之间不需要"另一方面""相比之下"——直接换段，靠语义对比让读者自己接上。

---

### 9. 人物画像：极短篇幅，靠 1-2 个具体细节立人

✅ W1 朱骏（豆包负责人）：
> 豆包的负责人是朱骏（Alex），他曾因一次在火车上捕捉到了年轻人的社交方式变化，后来做出了在美国走红的短视频产品 musical.ly。熟悉他的人形容他像 "互联网诗人"，他常在飞书签名里写下当下心境——有时自比 "南柯太守"，借典故表达对功名如梦的感受；有时则随手分享最近读到的一本小说。

（一段话立起一个人——经历 + 标签 + 具体行为细节。"飞书签名""南柯太守"是关键，没有这两个具体物，画像就垮。）

✅ W3 向晓晗：
> 本科在中文系，硕士读了一个 "全球事务" 专业，从零开始学投资，晓晗入行六年，本身也是抱着做一年是一年的心态，"我觉得我没有被教化，本来也不是金融行业出身，对于 VC 这个行业我没有严肃感。"

**值得偷的**：写到一个真人时（创业者、CTO、投资人），用 **2-3 句话+ 1-2 个具体细节（行为、习惯、口头禅、签名、办公环境）** 立起来。不写"他是一个聪明、勤奋、有远见的人"——抽象形容词让人立不起来。

---

### 10. 类比必须有锚——具体的物 / 历史 / 物理含义

✅ W1: "中途岛战役" — 二战太平洋战争转折点。贯穿全文，结尾还回扣"巴别塔"。
✅ W4: "造梦工厂" — 不是泛指，是景峰真的给自己影视基地起的名字。
✅ W6: "蒸汽机已经被发明了出来，但它有时候跑得还没有马车快。" — 工业革命具体物。
✅ W6: 自动驾驶 5 小时无接管的拐点 → 类比 token 消耗增长曲线（作者本人做过自动驾驶，是真经历，不是百度的）。

❌ 反面（AX 容易写到的）："AI 像一个全能实习生" / "Agent 像一个数字员工" / "大模型像水电煤"

**值得偷的**：类比要么从**历史/战争/物理实物**里取（中途岛、巴别塔、蒸汽机），要么从**作者真做过的事**里取（孟醒做过自动驾驶 → 用 5 小时无接管做类比）。绝不用其他人的二手类比（"全能实习生"是 OpenAI 演讲用滥的）。

---

### 11. 对立面：永远讲代价 / 张力 / 反方处境

✅ W4 整篇贯穿：
- 红果方法论的成功 → "免费、自己爽、无限效率"
- 同时讲：43 岁导演猝死 / 张宽要裁员 / 景峰怀念 2012 年慢工的孔雀
- 结尾："他们争夺的，都是人的时间，而人正越来越诚实地追逐自己的欲望"

✅ W5 全篇结构 = "巨大估值想象" + "巨大风险" 双线交织：
- 1 万亿估值故事 / 猎鹰 9 号毛利 70% / 星链现金流
- + 团队撕裂 / 9/12 联合创始人离开 xAI / FBI 介入 / 巴西最高法院 / 邻居反对数据中心

✅ W6: 全员 token-maxxing 兴奋 + 酸橙树/暗杀名单的恐惧。两端同时讲。

**值得偷的**：写一个东西"厉害"，**必须同时写它的代价 / 反对方的真实处境 / 它没说的部分**。不是为了 balance 而 balance——是因为现实里这俩永远共存。AX CLAUDE.md 里"对立面的理解与承认"原则，晚点是怎么落地的看这里。

### 12. 收尾：升华到大命题，但通过引用 / 留白 / 个人愿景，不是道德教化

✅ W1 收尾：
> "把公司当作能够单方面塑造世界的力量，本身就是危险的。" 达里奥·阿莫代伊说。

（引用 Anthropic CEO 一句话，不再多说。）

✅ W4 收尾：
> 不变的规则是，他们争夺的，都是人的时间，而人正越来越诚实地追逐自己的欲望。

（一句话总判断，留白。）

✅ W6 收尾：
> 这半个月我看到了那么多 "跟不上"，这确实让人焦虑。但如果 AI 真的在几年内让癌症变成慢性病、让材料科学快进二十年，那么这一场 "跟不上"，可能是人类发展历史上最大的一次提速。
>
> 我家宝宝今年两岁，明年可能会有第二个孩子，他们这一代要面对的那个世界什么样，我现在完全没有想象力去构建。
>
> 但我希望，在他们长大的世界里，多一些因 AI 而被治愈的人，而不是有更多燃烧瓶和枪声，砸向 AI 从业者的家门口。

（个人愿景做收尾，孩子做 anchor。情绪是真的，因为前文铺过。）

❌ 晚点绝不写的收尾：
- "总而言之，AI 时代充满机遇与挑战"
- "让我们拭目以待"
- "未来已来，你准备好了吗？"
- "时代的浪潮中，每个人都是参与者"

**值得偷的**：3 种合法收尾：
1. **引用收尾** — 找一句别人的话做点睛（行业内人士 / 哲学家 / 历史名言）
2. **判断留白** — 一句精炼的总结，戛然而止
3. **个人愿景** — 用 anchor 做（孩子、自己未来要做的事），不是空喊"我们要拥抱未来"

## 反面：晚点不做的事

| AI slop / mp 也禁的 | 晚点是怎么处理的 |
|---|---|
| 然而、综上所述、首先其次最后 | 直接换段，让语义对比 |
| "在 AI 快速发展的时代" | 上来就给具体场景/数字/人 |
| 大段加粗 | 几乎不加粗，靠段落短和单句段做强调 |
| "本文将分析" | 没有 meta 描述自己要写什么 |
| "你有没有这种感觉" / 假设性场景 | 全部是真实事件 / 真实数字 |
| "细思极恐" / "更狠的是" / 营销号情绪词 | 冷静陈述事实，让事实自己 punch |

## 应用到 AX 写作

### AX 写哪类文章可以套晚点结构

✅ **行业判断 / 趋势观察** — 比如 "Agent Economy" / "Why most AI startups will fail" 这类。**直接套 W1 / W4 / W6 的结构**：副标题做 thesis → 4-6 个 header（判断式/动作式）→ 每章节铺事实+数字+引用，最后一句下判断。

✅ **公司/产品深度拆解** — 比如写 OpenAI / Anthropic / Mio 的某个机制。**直接套 W5 结构**：4-5 个 header 围绕一条主线（赚钱机制 / 风险点）层层推进。

✅ **一线观察 / 出差日记** — 比如 AX 在 Compute Labs / Mio 的实战观察。**套 W6 结构**（专栏）：第一人称 + 极具体场景 + 段尾判断。**这种最适合 AX 博客的姿态**——builder 在跟同行喝酒。

### 不能直接套的部分

❌ **匿名信源** — "一位接近字节高层的人士告诉我们" 是记者专属，AX 写自己的产品/朋友/经历直接说"我"或"X 是我朋友"。

❌ **集体 "我们"** — W1-W5 用 "据我们了解" 是机构口吻。AX 是个人博客，用 "我" 即可。

❌ **多人物群像** — W1 同时跟踪字节/阿里/腾讯 3 家是记者深度报道才负担得起的工作量，AX 写一个 thesis 不需要这种 scope。

### 与现有 zh-writing-style-reference.md 的关系

`zh-writing-style-reference.md` = 抽象规则（句长、English 比例、kill list、HKR 自检）。**这份文件 = 具体范文样本**。两个一起用：
- **写之前**：先读这份范文，校准语感
- **写之中**：按 zh-writing-style-reference.md 的规则下笔
- **写之后**：用 zh-writing-style-reference.md 的 4 层 checklist 自检

如果两份文件冲突，**以这份范文为准**——规则是从范文里抽出来的，范文比规则更原始。

## 风格 quick-reference（写之前看一眼）

```
开头        = 具体人 + 具体动作 + 具体时间 + 一句反常判断
副标题      = thesis 一句话拍出来
小标题      = 判断式/动作式/反差式，绝不描述性
段落        = 2-4 行，单句段做转折
数据-观点比  = ≥ 3:1，每段必须有 anchor
判断位置    = 段尾，不是段首
转场        = "但""更""到了 2025 年"，禁"然而""综上所述"
引用        = 真实出处，AX 写自己的就用"我"
人物画像    = 2-3 句 + 1-2 个具体细节（行为/习惯/口头禅）
类比        = 历史/战争/物理实物，或作者真做过的事，禁二手类比
对立面      = 讲代价、讲反方真实处境、讲没说的部分
收尾        = 引用 / 判断留白 / 个人愿景，禁道德教化
```
