# 6 XHS Copywriting Styles for Jewelry

Each `analysis.json` contains 6 copy bundles — one per style. Each bundle has:
- 5 hooks (titles, ≤20 chars, with 1-2 emoji each)
- 5 selling points
- 1 full XHS post (300-500字)
- 5 hashtags

## Style matrix

| Style | Persona | Tone | When to use |
|---|---|---|---|
| **闺蜜种草** | Bestie sharing a find | "姐妹们！" "救命！" "宝藏好物" | 主推日常，转化率最高 |
| **专业测评** | Domain expert | 4C / GIA / 性价比 / 鉴定 | 高客单（钻石、彩宝、奢侈品）|
| **情绪叙事** | Self-care content creator | 文学意境，犒劳自己 | 高消费力女性自购 |
| **穿搭攻略** | Outfit influencer | 通勤/约会/度假场景方案 | 配饰类（项链、耳环、手链）|
| **文化叙事** | 国风/匠心 KOL | 东方美学 / 传承 / 留白 | 古法金、翡翠、新中式系列 |
| **送礼仪式感** | Gift narrator | 七夕/母亲节/520/生日 | 节日转化高峰期 |

## Tone rules per style

### 1. 闺蜜种草 (Bestie Recommendation)
- ❌ 广告腔："本店产品" / "品牌严选"
- ✅ 群聊感："姐妹们" / "我跟你们说" / "我前几天逛 XHS 看到"
- ✅ 用词：宝藏 / 小众 / 氛围感 / 闭眼入 / 安利 / 私藏

### 2. 专业测评 (Expert Review)
- 数据化表达：克拉数 / 净度 / 切工等级 / GIA 编号
- 结构：身份背书 → 规格详解 → 多维打分 → 优缺点 → 同价位对比 → 推荐人群
- 用词：测评 / 避雷 / 干货 / 硬核 / 4C / 性价比 / 鉴定

### 3. 情绪叙事 (Emotional Storytelling)
- 场景驱动：午后的窗光 / 周年纪念 / 一个人逛街
- 不直接卖货，靠情绪共鸣
- 用词：犒劳自己 / 仪式感 / 治愈 / 松弛感 / 小确幸 / 悦己

### 4. 穿搭攻略 (Styling Guide)
- 必须给具体方案：通勤白衬衫×细链、约会V领×珍珠、度假棉麻×古法金
- 不要泛泛"百搭"
- 用词：百搭 / 叠戴 / 混搭 / 点睛 / 通勤 / 约会

### 5. 文化叙事 (Cultural Heritage)
- 文化锚点 → 设计哲学 → 工匠故事 → 当代承接
- 语言要有文学功底，不是堆砌"东方美学"四个字
- 用词：留白 / 意境 / 传承 / 匠心 / 风骨 / 寓意

### 6. 送礼仪式感 (Gift Ritual)
- 切入：男友 / 老公 / 女儿 / 闺蜜送
- 结构：场景 → 选择过程 → 拆礼瞬间 → 收礼人反应 → 推荐预算方案
- 用词：仪式感 / 心意 / 不会出错 / 闭眼入 / 惊喜

## Pairing strategy

Each marketing image template has natural copy pairings:

| Image | Best copy style | Why |
|---|---|---|
| `wristNeck` (上手图) | 闺蜜种草 / 情绪叙事 | 直接看效果，激发"我也要" |
| `gemCutDetail` (微距) | 专业测评 | 显示工艺质量，配数据化文案 |
| `meaningCraftPoster` (大字报) | 文化叙事 | 工艺/寓意已在图上，文案延伸 |
| `giftScene` (送礼) | 送礼仪式感 | 节日营销标配 |
| `wearStyleGrid` (四宫格) | 穿搭攻略 | 4 套搭配方案直接展示 |
| `priceAnchor` (价格) | 闺蜜种草 | "救命她家也太便宜了" |
| `starTestimonial` (大字报) | 情绪叙事 / 闺蜜种草 | 高情绪强度 |

## A/B testing strategy

商家从一份 `analysis.json` 可以拿到 6 篇文案。建议：
1. 同一商品，每周发 2-3 个不同风格
2. 看 7 天后哪个风格点击/收藏率最高
3. 该 SKU 后续投放就锁定那个风格
4. 不同 SKU 适合不同风格 — 古法金大概率是文化叙事赢，钻戒大概率是送礼仪式感赢
