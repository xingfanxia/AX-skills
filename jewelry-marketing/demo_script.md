## Q&A 准备（路演现场，2 分钟问答）

预判评委可能问的问题 + 一句话答案，提前演练：

**Q1: 为什么是珠宝品类，不是更大盘子比如美妆？**
> 珠宝是 SKU 多、客单价高、试错成本最大的品类。一个钻戒 SKU 在小红书测试 3-5 个封面才能跑出爆款，每次外包 ¥1000+ 是真痛点。美妆有大量现成模板，珠宝没有 — 这是 niche 但深的护城河。

**Q2: 12 个营销模板的 prompt 是怎么来的？**
> 来自识川（shichuan）生产系统积累的研究 — 千瓜 2024 / 数英 / 我是产品经理 对周大福古法金、HEFANG×刘亦菲、樱桃珠宝 4.7W粉、阑珊珠宝 2.2W赞 等爆文的拆解。每个 prompt 都对应一个验证过的爆款公式，不是 AI 想象的。

**Q3: 跟 Midjourney / 通用 AI 工具差在哪？**
> 三个差异：(1) 材质感知 — 钻石高对比侧光、珍珠单光、翡翠均匀透光，自动注入 prompt；(2) 佩戴位智能 — 项链颈部、戒指手部、耳环侧脸；(3) 占位符纪律 — 价格 ¥___ 销量 [X]+ 永不写死，商家可直接发布不用 PS 改字。

**Q4: 设计 pipeline 的 sketch canonical 是什么意思？**
> 散石照输入后，先生成一张水彩+铅笔手绘稿作为 canonical 设计，其余 7 张（3D 渲染 / 佩戴 / CAD / 灵感板）都把这张 sketch 作为 reference image，保证 8 张图展示的是同一件设计 — 不是 8 个独立设计。这是 shichuan 验证过的两阶段方案。

**Q5: 商业化路径？**
> 三种：(1) 直接卖 skill 下载权（拍卖环节）— 1999-3999 元/份合理；(2) SaaS 化封装成网页工具，珠宝商家月订阅 99-299；(3) 接入小红书等平台的商家工作台，按调用量分成。短期最快是 (1)。

**Q6: 拓展性？只能做珠宝？**
> 当前珠宝聚焦是为了深度。同样架构可复用到：箱包配饰（核心是工艺微距 + 上手图）、文玩（同色系氛围板 + 工艺解读）、家居小物（场景化 + 一物多搭）。每个细分领域换 prompt 集合就可以，infra 不变。已经在识川系统里跑通了 14 个 subcategory × 12 模板 = 180 prompts。
