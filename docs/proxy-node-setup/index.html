07 / 09
五个静默失败点,全部已固化进 runbook

踩坑精选

A · 端口被封

边缘封 80/443

nginx 多监听高位端口 + 橙云 + Origin Rule 把回源改写到 2096,订阅 URL 保持干净 443。

B · CF token 误报

别用 /verify

/user/tokens/verify 对 zone 范围 token 会误报 Invalid;改用一次真实 zone 读取来验。

C · 网卡名

是 ens17

老教程默认 ens3,流量统计就空了;用 ip route 探真实网卡。

D · 证书双路径

两边要同步

nginx 读 ygkkkca 目录,sing-box 读 /etc/s-box/cert.pem;reloadcmd 同步两边。

E · UDP 误报

Hy2 / TUIC 是 UDP

用 nc -z(TCP)测会误报 filtered;改 nc -zuv 才准。
