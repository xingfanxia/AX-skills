# 世界观画布（Worldview Canvas）

> **怎么用**：这是开写前需要的**最小世界观（MVP）**。只填这 6 个字段就够开始写故事了，其余设定等剧情需要时再长。
> **填写区在每个字段的 `→` 后面；底部「填写示范」只是参考（换了题材，请勿照抄）。**
> **铁律**：世界为故事服务，不是反过来。每写一条设定，问自己——它是否直接影响某个角色的决定、制造一个剧情障碍、或解释一个看得见的后果？不能，就先别建。
> （方法论详见 `references/03-worldbuilding.md`）

## 📝 填写示范（题材：潮汐邮局 · 请勿照抄）
| 字段 | 示范内容 |
|---|---|
| 核心前提 | 话语与心意必须趁"对的潮汐"投递，否则会随退潮永远溶解消失 |
| 地理 | 一座只在特定潮汐才能通行/投递的环礁岛，岛心是一座高塔邮局 |
| 紧张态势 | 邮政总局要按"时效"给信定价（迟到的信廉价处理甚至销毁）；"守潮人"主张有些信值得等下一个潮汐 |
| 力量规则 | 邮差能"听见"未投递信里的心意；但每读一封，自己会失去一段记忆 |
| 主角阶层 | 岛上最准时、从不误点的铁面邮差长 |
| 文化禁忌 | 拆开或朗读不属于自己的信 |

推导链（示范节选）：信件可溶解→"时效"成硬通货→穷人的信永远排在退潮后→"能等下一个潮汐"成了特权→谁掌握潮汐表谁掌握人心。**自洽，不需要第二个谎言。**

## 防自嗨检查（交付前过一遍）
- [ ] 6 个字段都填了，且能开写
- [ ] 核心前提推出了至少 3 条不同层面的后果
- [ ] 没有在写"用不到的设定"——每条都服务于剧情/人物/可见后果
- [ ] 背景笔记字数没有超过正文（超过 = 世界观拖延症，停下来去写故事）

> **二游补充**：世界观靠"冰山"分层呈现（主线必读 / 好奇者探索 / 狂热者考据三层），核心大谜团长期不解做留存"北极星"。但冰山必须有露出水面的尖角——在游戏里持续给线索，而不是自己脑补一堆不呈现。详见 `references/03` §6 与 `references/05` §6。
