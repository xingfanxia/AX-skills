# 核心理念（主题）工作表

> **怎么用**：找到这个故事的**脊柱**——一句话主题（controlling idea），不是一个词。它将用来检验后面每一场戏"该不该留"。
> **填写区在每个 `→` 后面；底部「填写示范」只是参考（换了题材，请勿照抄）。**
> （方法论详见 `references/01-story-architecture.md` §1 与 `references/05-erciyuan-cn.md` §1 的"米哈游主题引擎"）

---

## 第一步：从模糊想法逼出主题（诊断问题）
逐个回答，矛盾的答案也留着：
1. 故事结束时，主角**懂得了**什么他一开始不懂的？ → 〔填写〕
2. 那个"懂得"，**对人生/人性说了什么**？ → 〔填写〕
3. 这个改变是**得到**还是**失去**？（正向/负向弧）→ 〔填写〕
4. 我希望玩家玩完后，**感受到/相信**什么？ → 〔填写〕

---

## 第二步：写出主题句（McKee 格式）
`[价值] 因为 [原因] 而 [被赢得 / 被摧毁 / 被守住]`

→ **我的主题句**：〔在此填写〕

**警告**：如果你写的是"我的游戏是关于*情感/自由/记忆*的"——那是**主题词**，不是主题句。逼自己说出：这个故事**对**它**说了什么**。

---

## 第三步（二游强烈推荐）：米哈游主题引擎
不要从"我想做一个 XX 设定的故事"出发，要从**"当代年轻人正被什么真实情绪困住"**出发：
1. **当代真实情绪**（孤独 / 内卷 / 记忆与遗忘 / 自我价值 / 命运无力感 / 情感羞耻 …）→ 〔填写〕
2. **拆给若干角色各背一份不同的理解** → 〔填写〕
3. **给一个比现实更理想的解**（不是廉价大团圆，是"牺牲中留希望"）→ 〔填写〕

---

## 📝 填写示范（题材：潮汐邮局 · 请勿照抄）
- 主角懂得了：有些信必须自己投，哪怕迟了十年、哪怕会溶解。
- 对人性说了什么：迟到的真心不是不值钱，是没人敢在对的潮汐说出口。
- 得到还是失去：失去（那封信溶了），但被听见了。
- **主题句**：迟来的真心因为错过潮汐而几乎永远沉默，却因为有人甘愿再等一个十年的潮汐而被听见。
- 当代真实情绪：我们总把"想说的话"拖到"更好的时机"，结果时机永远不来。
- 拆给角色：邮差长信"准时投好别人的就行"；总局信"过期就该贬值"；守潮人偏要为一封信等下一个潮汐。
- 更理想的解：主角终于自己投出那封迟到十年的信——信溶了，但收信人此刻就站在塔下听见了。

## 酸性测试（交付前必过）
- [ ] 我能用一句话说出主题句（不是一个词）
- [ ] 如果把结局改成相反（主角失败而非成功），故事**不**还成立——说明主题已长进结构，不是装饰
- [ ] 主题没有被角色"直接说教"出来，而是靠事件本身论证
- [ ] （二游）主题源自一个当代真实情绪，且被拆给了多个角色各自诠释
