# 可抽角色价值栈（Character Value Stack · 10 层）

> **怎么用**：二游角色同时是**玩法单位 + IP 商品 + 剧情驱动 + 抽卡动机**，四向必须指向同一处。这份 10 层检查表把"角色设计 × 商业"拆成 10 个可独立验收的问题——**从上往下填；填不出来的那层，就是这个角色当前最薄弱的环节**。
> 它**配合**人物档案用：人物档案（`character-sheet-template.md`）管角色"好不好"（伤口-谎言-弧线/声音）；价值栈管角色"能不能在商业机器里持续供货"。
> （方法论详见 `references/11-production-liveops.md` §2；反差萌/防玛丽苏见 `references/02` §6；亲密/塌房一致性见 `references/10` 萌·恋）

---

- **1 视觉锚点**：一眼让人记住什么？（剪影 / 配色 / 武器 / 标志性表情或动作）→ 〔填写〕
- **2 身份锚点**：TA 在这个世界里是谁？（职业 / 阵营 / 一个秘密）→ 〔填写〕
- **3 能力幻想**：玩家抽到后幻想获得什么？（战斗方式 / 技能主题 / 玩起来的爽点）→ 〔填写〕
- **4 性格矛盾**：矛盾在哪？（外冷内热 / 强大却脆弱…）**⚠️ 必须有伤口支撑**，否则就是两个标签缝在一起。→ 〔填写〕
- **5 关系钩子**：和谁有未解决的关系？（宿敌 / 旧友 / 师徒 / 误解 / 血亲）→ 〔填写〕
- **6 剧情功能**：为什么 TA**必须**出现在这个版本？（线索 / 钥匙 / 反方 / 受害者 / 见证人）→ 〔填写〕
- **7 拥有回报**：抽到的玩家额外获得什么情感？（专属语音 / 个人剧情 / 特殊台词 / 待机互动）→ 〔填写〕
- **8 未拥有体验**：没抽到的玩家是否仍能理解 / 喜欢 TA？（主线表现 / 试用关 / 活动 / NPC 互动）→ 〔填写〕
- **9 社区传播点**：玩家会截图 / 剪辑 / 二创什么？（名台词 / 名场面 / 反差 / CP / 梗）→ 〔填写〕
- **10 长线空间**：未来还能写什么？（过去 / 阵营 / 变体 / 觉醒 / 关系进展）→ 〔填写〕

> **两条最关键的提醒**：
> - **第 7 层 vs 第 8 层是二游特有的双重责任**：抽到的玩家要有专属回报（否则花了钱没"到账"），没抽到的玩家也要能理解并喜欢 TA（否则复刻没人想抽）。**这两层不能用同一段内容应付。**
> - **第 4 层（性格矛盾）必须能被一个伤口解释**——为什么这个凶人怕黑？因为伤口。没有伤口的反差是为搞笑而搞笑。

## 📝 填写示范（题材：潮汐邮局 · 角色"闻汐" · 请勿照抄；下面只摘录部分层做示范，实际请 10 层全填满）
- 1 视觉锚点：永远别在腰间的黄铜潮汐表 + 一封从不投出、边角磨白的旧信。
- 3 能力幻想：掐着潮汐"读时投递"——技能在精确时点触发最大效果，玩起来是"算准节拍"的爽。
- 4 性格矛盾：全岛最铁面、零误点的邮差长 × 自己那封信攥了十年没敢投（根植于"没在对的潮汐投出"的伤口）。
- 6 剧情功能：1.0 主危机"灰潮吞信"的唯一知情者 + 教学向导。
- 7 拥有回报：拥有后解锁她的待机语音"……这封，再等等"，与一段她独自走到潮线又折返的个人小剧情。
- 8 未拥有体验：主线里她当众投出一封"本该她自己的信"的高光场面，没抽到的人也会记住她。
- 9 社区传播点："还有两刻退潮"的报时口癖 + 那封旧信的特写（玩家爱猜收信人是谁）。
