# 二游版本故事 + 卡池协同 + 长线路线图（Version & LiveOps）

> **怎么用**：二游不是"写完一个故事"，是"按版本持续供货"。这份表帮你把**单个版本怎么讲**和**长线怎么排**落到纸上。三块：① 单版本故事卡（每版填一张）；② 卡池-叙事协同（每个上线角色填一行）；③ 长线路线图（几版一起看）。
> **填写区在每个 `→` 后面；底部「填写示范」只是参考（换了题材，请勿照抄）。**
> （方法论详见 `references/11-production-liveops.md` §3/§6；活动四象限见 `references/05` §10.C；登场排期见 `references/05` §10.B）

## 一、单版本故事卡（每个版本至少推进四条线：① 主线 ② 角色 spotlight ③ 玩法/活动 ④ 长线 Hook）

- **版本号 / 主题** → 〔填写〕
- **版本一句话**（这一版到底在讲什么 / 给玩家什么）→ 〔填写〕
- **主推角色** → 〔填写〕
- **角色登场理由**（TA 为何必须在这一版出现？= 价值栈第 6 层）→ 〔填写〕
- **主线推进了哪一步**（把整体故事往前挪了多少）→ 〔填写〕
- **本期情绪基调**（热血 / 治愈 / 悬疑 / 悲伤 / 荒诞 / 节庆 / 恐怖）→ 〔填写〕
- **新地点 · 新玩法** → 〔填写〕
- **活动结构**（大活动套小活动？叙事比重 × 可重复性象限）→ 〔填写〕
- **中段反转** → 〔填写〕
- **高潮事件** → 〔填写〕
- **结尾 Hook**（留给下一版的钩子）→ 〔填写〕
- **抽卡配合**：角色池 → 〔填写〕 / 武器·音擎池 → 〔填写〕 / 试用关卡（让非拥有者体验手感）→ 〔填写〕 / 剧情展示（角色的决定性场面）→ 〔填写〕
- **新玩家补偿**（对新入坑玩家是否友好 / 有无补足前情）→ 〔填写〕
- **老玩家伏笔回报**（这一版回收了哪个之前埋的坑）→ 〔填写〕

**填表原则**：1.0 先把核心体验立住（别首发就埋一堆看不懂的坑）；每版"本期满足 + 长期未解"（双重收尾）；每个角色都有登场理由；**每 3–4 版回收一个中型伏笔**（信任建在节奏上）；周年承担情绪峰值或世界推进；复刻/常驻化进叙事规划。

## 📝 填写示范（题材：潮汐邮局 · 1.0 版本 · 请勿照抄）
- 版本号/主题：1.0「第一封信」
- 版本一句话：玩家成为潮汐邮局新差，送出第一批"会随退潮溶解"的信，初识"信比人活得久"的规则。
- 主推角色：闻汐（铁面邮差长）
- 角色登场理由：她是唯一能教玩家"读潮汐投信"的人，也是 1.0 谜团（她攥着的那封旧信）的核心。
- 主线推进：建立"信的世界"基本规则 + 抛出"灰潮在吞噬旧信"的主危机。
- 本期情绪：治愈打底，结尾一点怅惘。
- 结尾 Hook：闻汐袖中那封从不投出的信，特写一闪——更大谜团（她在等谁）留下。
- 卡池配合：闻汐池 + 试用关"末班潮投递"让非拥有者体验她的报时投信手感。
- 老玩家伏笔回报：（1.0 无可回报，是埋坑版）；下版起每 3 版回收一封"未送达的信"。
