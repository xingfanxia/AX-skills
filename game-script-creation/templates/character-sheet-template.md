# 人物档案（Character Sheet）

> **怎么用**：人物档案不是户口本，是**决策引擎**——当你不知道"这个角色此刻会怎么做"时，回来看它就有答案。**先填"核心"，再填其余。** 填不出"一句话核心特质"而只会写形容词，就说明你还不了解这个角色，回去想伤口。
> **填写区在每个 `→` 后面；底部「填写示范」只是参考（换了题材，请勿照抄）。**
> （方法论详见 `references/02-character-craft.md`；二游角色见 `references/06-erciyuan-jpkr.md` §2）

---

## 核心（决策引擎，先填这块）
- **一句话核心特质**：用"X 是那种会 ___ 的人"造句，填**行为**，不是形容词。→ 〔填写〕
- **伤口（Wound / Ghost）**：过去具体发生了什么，打碎了什么？（要具体）→ 〔填写〕
- **谎言（Lie）**：从伤口长出的错误信念。→ 〔填写〕
- **渴望（Want，外部目标）**：相信这个谎言的人会去追什么？→ 〔填写〕
- **需要（Need，内在真相）**：要真正痊愈必须接受的真相（几乎是谎言的反面）。→ 〔填写〕

## 弧线（这个故事里）
- **弧线类型**：正向改变 / 平稳(steadfast) / 负向（堕落·幻灭·沉沦）→ 〔填写〕
- **结束状态**：他在故事结尾变成了什么样？→ 〔填写〕
- **伤口怎样被剧情正面攻击**：（光在背景里提一句不算弧线）→ 〔填写〕

## 声音与行为（让角色可被"听见"）
- **说话方式**（正式/简短/啰嗦/爱用反问/用幽默当盔甲…）→ 〔填写〕
- **口头禅 / 第一人称 / 怎么称呼"你"** → 〔填写〕
- **从不直说的事**（他回避/委婉/沉默掉什么？）→ 〔填写〕
- **不设防时露馅的瞬间** → 〔填写〕

## 矛盾（让角色立体）
- **对外表现** → 〔填写〕  **真实的他** → 〔填写〕  **裂缝在哪露出** → 〔填写〕

## 二游专属字段（角色即商品时填）
- **3 秒抓人的"拉取钩子"特质** → 〔填写〕
- **反差萌(gap moe)方向**：`[表层特质] + [与之尖锐矛盾、且根植于伤口的内里]`。反差必须能被伤口解释。→ 〔填写〕
- **可被演出的"声音"**（语速 / 语气词 / 第一人称 / 怎么称呼你，为声优而写）→ 〔填写〕
- **登场规划**：T-N 埋伏笔 → T-1 高光铺垫 → T（卡池）个人剧情 + PV（见 `references/05` §10.B）→ 〔填写〕
- **角色 PV 文案思路**：史诗第三人称定调 → 第一人称对玩家亲密直述 → 留一句钩子 → 〔填写〕

## 📝 填写示范（题材：潮汐邮局 · 角色"闻汐" · 请勿照抄）
- 一句话核心特质：闻汐是那种能掐着秒投出全岛每一封信、却把自己十年前那封攥在手里、从没敢投的人。
- 伤口：十年前她没在对的潮汐投出给某人的信，那人随退潮离岛，再没回来。
- 谎言：只要把别人的信都准时投好，自己那封就不必面对。
- 渴望：成为零误差的邮差长，掌管全岛潮汐投递。
- 需要：有些信必须自己投，哪怕迟了、哪怕会溶解。
- 弧线：正向改变 / 结尾：终于自己投出那封信，信溶了，但收信人听见了。
- 反差萌：全岛最铁面、从不误点的邮差长（表层）× 自己攥着十年没敢投的信（内里，根植于伤口）。
- 声音：自称"本差"，爱用潮汐报时（"还有两刻退潮"），称玩家"新来的"，很久后才改口叫名字。

## 交付前自检
- [ ] "一句话核心特质"是行为不是形容词
- [ ] 伤口→谎言→渴望→需要链完整且自洽
- [ ] 有一个**真的会让他付出代价**的缺点（不是"笨拙得可爱"这种伪缺点 → 防玛丽苏）
- [ ] 盖住名字，光看台词能认出是他在说话（声音可区分）
- [ ] （二游）反差与伤口挂钩，不是为搞笑而搞笑
