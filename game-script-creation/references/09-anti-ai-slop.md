# 09 — 反 AI 味 / 反套路检查清单（剧情向）

> **用途**：在宣布任何一段剧情、人物或对白"写好了"之前，过这份清单。它专治游戏/小说叙事里的"套路感"与"AI 味"。综合自 `01`–`08` 的 craft 研究 + 创作社区公认的新手/AI 雷区。
> **给 AI 教练的话**：这不是给创作者背诵的规则，是你审稿时的 pattern-match 列表。发现命中项，用"前/后改写"示范怎么修，而不是只报错。

---

## 1. 对白的 AI 味（最致命）

| 雷区 | 长什么样 | 修法 |
|---|---|---|
| **把话说死（on-the-nose）** | "我很生气，因为你没打给我。" 角色直接报出情绪+原因 | 写潜台词："你的手机是用来看时间的吗？" |
| **"你知道的，鲍勃…"信息倾倒** | 角色互相讲彼此都知道的设定，只为告诉玩家 | 把信息藏进冲突里喂；或借一个真不知情的新人之口 |
| **人人一个声音** | 盖住名字分不清谁在说 | 给每个角色不同用词/句长/打断方式/第一人称；改稿时把一个角色的台词集中起来读 |
| **解释自己的潜台词** | 写完一句潜台词，又补一句旁白把它解释了 | 删掉解释。相信读者 |
| **万能脏话/语气词当性格** | 靠"哼""啧""切"假装有个性 | 性格来自他**说什么内容和怎么回避**，不是语气词 |

## 2. 人物的套路

| 雷区 | 长什么样 | 修法 |
|---|---|---|
| **玛丽苏/全能主角** | 美丽聪明万人迷、从不真正失败、缺点是"太善良" | 给一个**真会让他付出代价**的缺点；让他失败并挣回来 |
| **形容词人设** | "她勇敢、善良、坚强" | 换成行为+矛盾："她是那种会在葬礼上笑得最大声的人" |
| **有伤口没弧线** | 背景里有惨痛过去，但不影响当下任何决定 | 让谎言**现在**还在让他做错决定；让剧情正面攻击伤口 |
| **伪缺点** | "笨拙得可爱""太为别人着想" | 缺点要真的造成损失，伤到别人 |
| **反差萌悬空（二游）** | 反差只为搞笑，与角色无关 | 反差必须由伤口解释（为什么这个凶人怕黑？因为伤口） |
| **工具人配角** | 配角只为衬托主角存在 | 给每个重要配角一个与他人冲突的"渴望" |

## 3. 结构 / 情节的套路

| 雷区 | 长什么样 | 修法 |
|---|---|---|
| **"然后…然后…"** | 节拍只靠时间顺序串，没有因果 | 用"但是(复杂化)/所以(后果)"重连；删纯过渡场景 |
| **中段注水** | 中间像一串无关小冒险，赌注不升级 | 装一个真正的**中点翻转**；每次失败都让他失去点什么 |
| **天降解围（deus ex machina）** | 危机被一个临时冒出来的力量/巧合解决 | 提前埋好"契诃夫之枪"；高潮必须靠主角自己的选择解决 |
| **赌注空泛** | "拯救世界"，但玩家不在乎 | 先让玩家在乎一个**具体的人**，再升级到世界 |
| **悬念悬而不收（二游）** | 抛一堆神秘，后期不回收 | 伏笔账本登记每个坑的回收章；回收时要让人"啊原来如此" |
| **开场灌设定（二游翻车点）** | 开头就甩世界观名词、强行让 NPC 吹主角 | 先给钩子（战斗爽/酷角色/强意象），设定延后到玩家有动机听 |

## 4. 文笔的 AI 味

| 雷区 | 长什么样 | 修法 |
|---|---|---|
| **告知而非展示** | "她很紧张。" | 展示证据让读者自己感觉："她又一次把手机翻过来扣在桌上" |
| **滤镜词** | "她看见门开了，感到一阵恐惧" | 直接呈现："门开了。寒意顺着脊背爬上来" |
| **白屋症** | 对白漂在没有场景的虚空里 | 锚 2–3 个**起作用**的感官细节 |
| **紫色文风（堆辞藻）** | 形容词叠罗汉、辞典式炫技 | 一个强名词/动词 > 三个修饰语；朗读，删掉自我表演的部分 |
| **句子同一节奏** | 每句都中等长、主谓宾 | 长短交替；长句后扔一个三字短句制造冲击 |
| **跳头** | 一场戏里在多个角色脑内乱窜 | 一场一个 POV；别人的心理靠外部表现展示 |

## 5. 中文 AI 写作的高频雷词 / 雷句（直接搜删）

- **套话**：首先…其次…最后、综上所述、值得注意的是、不难发现、总而言之
- **空洞动词/名词**：赋能、抓手、闭环、维度、底层逻辑、心智、对齐颗粒度
- **高频踩雷**：说白了、本质上、换句话说、不可否认、这意味着、某种程度上
- **教科书开头**：在当今……的时代、随着……的不断发展
- **翻译腔/AI 腔**：让我们、不得不说、堪称、可以说是、无疑
- **空泛指代**：不说具体名字，只说"某个角色""一种力量""一些事情"
- **编造**：搜不到/想不通的设定，诚实标注"待定"，**绝不硬编**

> 注：这些是**叙事散文/旁白**里的雷区。角色对白若是角色本来就会这么说话，则不在此限——人物声音 > 规则。

## 6. 二游专项套路（额外）

- **演出墙 / 静态对话墙**：大段无演出的对话——关键情绪点必须配立绘/表情/镜头/BGM/语音。
- **角色突变无铺垫**：性格大转弯却没有铺垫 = 塌房前兆。
- **关键信息锁在限时活动**：新玩家读不到的主线必需信息，绝不能只放限时活动。
- **文化梗超载**：本地化等效不了的成语/谐音梗堆砌——面向全球要有意识控制或加译注。
- **卡池角色硬塞**：角色登场不是临时卖货，要提前 N 个版本用伏笔账本铺好叙事位置。
- **"自嗨式冰山"**：自己脑补一堆设定却不在游戏里露出尖角——冰山必须持续给线索。

## 一句话总检
> 删掉这一段/这个设定/这句台词，读者的理解或感受会变差吗？不会，就删。它在"描述怎么做"还是"做成什么样"？描述本身没问题，但**让事件和行为说话，别让旁白替它们表态**。
