## 三、这个 skill 能帮用户做什么

**一条全流程陪伴线**，每步交付一份实体文档，最终汇成「创作圣经」：

```
S0 校准 → S1 种子定调 → S2 世界观 → S3 核心理念(主题)
        → S4 主线框架 → S5 人物核心特质 → S6 具体场景脚本 → S7 修改
```

四个关键特性，对应全部需求：

1. **从模糊到具体的 full loop**：把"脑子里一个画面/感觉"一路引导到世界观、主题、主线框架、人物特质，再**辅助写出具体的主线与场景对白脚本**（不止做设定，能真正落笔成戏）。
2. **自适应水平**：开场用小样本判断真实水平（零基础 / 有想法没技巧 / 有直觉没训练 / 已能成稿），据此调旋钮——不把人当纯小白淹没，也不居高临下。
3. **中文为主、多语言支持**：全程中文交流/提问/示范/产出；术语先中文再括注英文。
4. **通用适配所有游戏类型，二游覆盖最深**：RPG/开放世界/视觉小说/Roguelike/模拟/解谜/竞技/沙盒都能走；二次元抽卡有最深专项。

**怎么用**：直接说"我想做个游戏剧情但不知从哪开始""帮我把这个点子做成游戏故事""做二次元抽卡手游想写主线/角色/活动剧情"即可触发。拿不准某步产出长什么样，看 `examples/worked-example-erciyuan.md`。
