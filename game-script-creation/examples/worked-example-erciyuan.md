# 完整范例：一个二游剧情从概念到场景脚本

> **这是什么**：用一个虚构的二次元抽卡手游《拾感纪》，把本 skill 的全流程**从头到尾走一遍**，展示每个工位产出长什么样、AI 教练如何随创作者水平调整。创作者拿不准某一步该交付什么时，对照这个看。
> 范例创作者设定为 **Level 1（有想法没技巧）+ 偏 pantser（边写边想）**——这是最常见的画像。

---

## S0 · 校准（教练内部判断，不念给创作者听）

创作者开场说："我想做个抽卡游戏，脑子里有个画面——一座靠'卖眼泪'运转的城市，特别想做这个氛围，但完全不知道怎么写成剧情。"

教练让她写 3 句话样本（"一个人收到坏消息的瞬间"）。她写：
> "他听到消息后非常震惊和难过，久久不能平静，心里充满了痛苦。"

**判断**：有画面感和热情（Level 1，不是 L0），但样本是"告知"不是"展示"、堆情绪形容词——典型 L1。她说"想先抓氛围"——偏 pantser。
**策略**：不堵她做大设定；从她最热的"卖眼泪的城市"入手（S1→S2），用"陪写(we-do)"在她自己的句子上做前/后改写，问题触发式教技巧。术语低密度。

---

## S1 · 种子与定调

教练问诊断问题，创作者答：氛围像"赛博朋克 + 旧上海的交易所"；故事关于一个给别人情绪定价的人；玩完希望玩家"重新想认真地为一件事哭一次"。
**种子陈述**：在一座把情绪当货币交易的城市里，一个麻木的"情感定价师"，因为一滴无法定价的眼泪，重新学会感受。
**游戏类型定位**：剧情向二游 → 查 `references/04` §6.3 五层叙事；查 `references/06` 选传统。

---

## S2 · 世界观画布（产出物）

| 字段 | 内容 |
|---|---|
| 核心前提 | 强烈情绪会结晶成可采集、交易、武器化的实体"心晶" |
| 地理 | 全城建在巨型"情绪交易所"上，越往下情绪越廉价，地底是被抽干情感的"灰区" |
| 紧张态势 | 交易所财阀想垄断高纯度"悲伤心晶"；底层"无偿者"组织主张情感不该被定价 |
| 力量规则 | "共鸣师"能感知结晶里的真实情绪、辨真伪；但每共鸣一次，永久磨损自己感受同种情绪的能力 |
| 主角阶层 | 交易所雇佣的中层鉴定共鸣师，靠磨损自己给别人情感定价 |
| 文化禁忌 | 当众流泪是最大失态——眼泪是"没卖出去的廉价货" |

推导链（节选）：情绪可实体化 → 成硬通货 → 穷人卖情绪过活 → "负担得起去感受"成特权 → 控制交易所=控制人心。**自洽，不需要第二个谎言。**

---

## S3 · 主题句（产出物）

- **当代真实情绪**：我们越来越羞于、也没空"真实地"表达情绪，把感受都外包/压抑掉了。
- **主题句**：真实的情感，因为被明码标价而几乎消亡，却因为有人愿意无偿地为另一个人流泪而被重新点亮。
- **拆给角色**：苏霁信"能定价就能掌控"；底层少年信"情绪是奢侈品，活下去要紧"；无偿者首领偏要无偿地哭。
- 酸性测试：若结局改成"苏霁成功垄断定价权、彻底麻木"——故事**不**成立（变成反讽悲剧需重写主题）。✅ 主题已长进结构。

---

## S4 · 主线框架（产出物，分形三幕 + 空宝箱）

- **空宝箱总目标**：玩家第一天就知道——集齐被财阀拆散卖掉的"七滴原初之泪"，才能让灰区重新有情感。共 7 个篇章。
- **第一篇章「定价师」**（教学用单弧，三幕）：
  | 锚点 | 内容 |
  |---|---|
  | 激励(~10%) | 苏霁接到一单：鉴定一滴"报不出价"的泪 |
  | 卷入(~20%) | 她发现这滴泪是三年前被她导师卖掉的、属于她自己的那滴 |
  | 中点翻转(~50%) | 财阀要销毁这滴泪以维持"悲伤稀缺"；苏霁从"乖顺的定价师"转为"偷泪者" |
  | 低谷(~75%) | 为带走泪，她必须共鸣它——而这会永久磨损她最后一点感受悲伤的能力 |
  | 领悟(~80%) | 无偿者少年替她共鸣，替她哭了那场她哭不出的哭 |
  | 高潮(~95%) | 苏霁当众流泪（打破最大禁忌），让交易所第一次"悲伤贬值" |
  | 结局(~100%) | 财阀垮了一角；但七滴泪还差六滴——**更高层留坑** |
- **双重收尾**：闭合（夺回自己的泪、苏霁会哭了）+ 留坑（还有六滴 / 导师为何当年要卖）。
- **伏笔账本**：导师动机 → 计划第 4 篇章回收。

---

## S5 · 人物档案（产出物，主推卡池角色）

**苏霁**（首章主线高光 + 首期卡池）
- 一句话核心特质：**苏霁是那种能在三秒内给你的眼泪报出市价、却三年没为自己哭过一次的人。**
- 伤口：唯一一次真情结晶的泪被信任的导师拿去卖了，换她晋升。
- 谎言：只要能定价，就能掌控情感、不会再受伤。
- 渴望：成为交易所最顶级评估师，掌握定价权。
- 需要：接受有些情感无价、且注定会失去——正因如此才珍贵。
- 弧线：正向改变。
- 反差萌：交易所最冷酷的定价机器（表层）× 自己患"情感失语"、三年没哭过（内里，根植于被背叛）。
- 声音：自称"在下"，爱用"这值……"给万物报价；称玩家"客户"，很久后才改口叫名字。
- PV 文案思路：第三人称定调（"她为整座城市的悲伤标价，唯独标不出自己的"）→ 第一人称亲密（"……你这滴，我破例，不收费"）→ 钩子。

## S7 · 修改（要点）

放凉 → 通读记笔记 → 反向大纲发现"委托人身份揭示太早"→ 结构 pass 把它推后 → 字句 pass 删掉初稿里"她感到一阵熟悉"（滤镜词+告知），改成动作"瞳孔缩了一下"。**先结构后字句，没在会被删的句子上抠字。**

---

## 这个范例展示了什么
1. 全流程 6 个产出物如何环环相扣（世界观的"心晶"→主题的"定价"→人物的"定价师"→场景的"报价腔"）；
2. AI 如何按 L1+pantser 调整（从她的热情入手、陪写、问题触发教学、低术语）；
3. 二游专项如何落地（空宝箱七章、双重收尾、反差萌、演出层、避开开场灌设定）。
> 你的项目不必照搬这个世界——它只是让你看见"填好之后长什么样"。
