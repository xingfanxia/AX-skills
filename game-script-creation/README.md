# game-script-creation

**从零陪伴非职业创作者完成游戏剧本的全流程 skill。** 把模糊的灵感一步步引导成世界观、核心理念（主题）、故事主线框架、人物核心特质，再辅助写出具体的主线/支线剧情和场景级脚本——从"概念定型"到"具体写作"的 full loop。

通用适配所有游戏类型（RPG / 开放世界 / 视觉小说 / Roguelike / 模拟 / 解谜 / 竞技 / 沙盒…），其中**二次元手游抽卡游戏（二游）覆盖最深**。

## 它为谁而做
为**没有任何编剧/小说写作经验、但有热情和点子**的创作者。skill 的核心是：**先校准创作者的真实水平**（零基础 / 有想法没技巧 / 有直觉没训练 / 已能成稿），再据此调整辅导力度——既不会把小白淹没在术语里，也不会对有直觉的人居高临下。**全程中文为主，多语言支持。**

## 怎么用
对 AI 说类似的话即可触发：
- "我想做个游戏的剧情，但不知道从哪开始"
- "帮我把这个点子做成一个游戏故事"
- "我做的是二次元抽卡手游，想写主线 / 角色 / 活动剧情"
- "我有个世界观/角色，怎么发展成完整剧本"

AI 会带你走 **8 站**（1 个校准站 + 7 个创作工位）：**校准 → 种子定调 → 世界观 → 核心理念 → 主线框架 → 人物核心特质 → 具体场景脚本 → 修改迭代**，每一步都交付一份你能带走的实体文档，最终汇成一份「创作圣经」。

## 研究基础（Research Foundation）

这个 skill 不是凭空写的。它建立在一轮**系统性深度调研**之上——8 个并行研究流，交叉验证、引用一手来源，把"优秀游戏案例 + 公认创作知识"蒸馏成 AI 可直接调用的可操作知识库（全部存于 `references/`，每篇附 Sources）：

|---|---|---|

| 03 | **世界观构建** | 桑德森魔法三定律、海明威冰山理论、"一个大谎言"推导链、Tolkien 次创造、游戏环境叙事（Dark Souls）|

## 设计原则
- **结果导向，不空谈**——每个工位都落一份实体文档。
- **自适应**——按创作者真实水平（靠样本判断，不靠自评）动态调整手把手程度、术语密度、"替写/陪写/让她写"的比例。
- **故事永远是创作者的**——AI 是脚手架，给框架、选项、示范、反馈，不替她拍板创意。
- **二游覆盖最深，但通用适配所有游戏类型**。
