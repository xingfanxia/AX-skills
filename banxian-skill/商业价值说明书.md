## 200 字简介（提交用）

半仙占卜 Skill 是即开即用的「赛博半仙」，开箱包含小六壬、梅花易数、六爻三套传统占卜术。底层是 panpanmao 玄学平台积累的真实算法引擎与专业知识库——64 卦完整数据、纳甲、六亲、世应一应俱全；外层是会用 KPI、房贷、相亲、裁员等当代词汇接住用户问题的 AI 半仙人设。

装上即用：用户描述事情 → 半仙智能选起卦法 → py 引擎出真卦 → 按卦象给针对性解读 → 同一桩事支持多轮追问。守"一事一占""医不问卦"等占卜规矩，敏感话题半仙化软处理含心理援助热线。算法与知识有真实占卜业务背书，不是 prompt 编的。

## 商业落地方向

### 1. 内容创作者一键供给玄学短视频 / 小红书图文

博主直接调用 skill，输入"用户问 X" → 输出半仙解读 + ASCII 卦象 → 配图发文。日产 10+ 条玄学内容零门槛。

### 2. 占卜小程序 / App 冷启动内核

跳过自研算法 + 知识库的 6 个月研发周期，skill 是即插即用的占卜服务后端，配 UI 即可上线 MVP。

### 3. 情感 / 决策辅助应用的氛围工具

心理咨询 / 婚恋 / 职业辅导类 App 嵌入半仙作为「氛围感」交互入口，提升留存和趣味性。

### 4. SaaS API 嵌入娱乐 / 心理类应用

按调用量计费的占卜 SaaS API，目标客户：直播打赏 / 陪聊 / 情感咨询 / 心理服务平台。

### 5. 玄学营销获客

品牌跨界 ——「测测你与 XX 品牌的缘分」「半仙告诉你今天该买啥」类传播 H5，背靠真实卦象数据，转化率高于纯娱乐占卜。

## 差异化优势

| 对比 | 普通 LLM 占卜 prompt | 半仙占卜 Skill |
|---|---|---|
| 算法真实性 | LLM 现编（容易 6 爻爻位算错） | py 引擎算，64 卦+纳甲表查表 |
| 知识专业度 | LLM 训练语料 | panpanmao 真实业务知识 |
| 邪修味 | 装神弄鬼 / 客服腔二选一 | 玄学底 + 当代穿插 + 专业骨气 |
| 合规性 | 容易翻车（算赌博/算寿） | 半仙式软处理 + 自杀热线强制 |
| 部署 | 需要 prompt 工程 | 即开即用，跨平台 |

## 技术指标

- **算法**：3 套引擎共 ~700 行 Python，21 个 pytest 单测全过
- **数据**：64 卦完整数据（卦辞 + 6 爻爻辞 + 纳甲 + 六亲）+ 6 神 + 12 时辰
- **知识**：13 个 markdown 文档（半仙 voice / 不占事项 / 5 大场景模板 / 解卦法）
- **依赖**：单依赖 lunardate，跨平台
- **大小**：< 1 MB ZIP（远低于比赛 10 MB 上限）
