# 体卦 / 用卦 / 互卦 / 变卦 解卦法

梅花易数解卦的核心方法——**体用互变**。

---

## 一、体卦 vs 用卦

| 角色 | 定义 |
|---|---|
| **体卦** | 不动的那个卦（上卦或下卦中没有动爻的） |
| **用卦** | 含动爻的那个卦 |
| 体卦 | = 我（求测人 / 主体） |
| 用卦 | = 彼 / 事 / 对方 / 客体 |

判断动爻位置定体用：
- 动爻在 1-3 爻（下卦）→ 下卦为用，上卦为体
- 动爻在 4-6 爻（上卦）→ 上卦为用，下卦为体

---

## 二、互卦（中段、过程、内里）

互卦取自本卦的中间四爻：
- **下互**：本卦 2-3-4 爻
- **上互**：本卦 3-4-5 爻

互卦表「**事情中段、过程、内里**」——即事情发展中你看不到的那一面。

---

## 三、变卦（之卦，最终走向）

动爻翻转 → 之卦。

之卦表「**事情结果、未来走向、最终归宿**」。

---

## 四、五行生克在卦象间（断吉凶核心）

判断**用卦五行 vs 体卦五行**的关系：

| 关系 | 判断 | 半仙断辞 |
|---|---|---|
| **用卦生体卦** | 大吉 | 「外援助我，事事顺遂——他人或事情助你成事。」 |
| **体卦克用卦** | 中吉 | 「我能制对方——你能掌控此事。」 |
| **比和（同五行）** | 平稳 | 「彼此和谐，无大吉亦无大凶。」 |
| **体卦生用卦** | 小凶 | 「你在泄气于人——付出不见回报。」 |
| **用卦克体卦** | 大凶 | 「外力压我——对方制约你，难以转圜。」 |

---

## 五、互卦也参与判断

完整解卦时，**体卦** 与 **互卦**、**变卦** 都要看五行关系：

| 互卦五行对体 | 含义 |
|---|---|
| 互生体 | 中段过程顺利 |
| 互克体 | 中段有阻 |
| 互比体 | 中段平稳 |

| 变卦五行对体 | 含义 |
|---|---|
| 变生体 | 最终结果向好 |
| 变克体 | 最终结果不利 |
| 变比体 | 最终保持现状 |

---

## 六、解卦顺序（半仙必须按这个顺序）

```
1. 看体用关系（生克比和）→ 大方向定吉凶
2. 看互卦（中段过程）→ 看事情会经历什么
3. 看变卦（最终走向）→ 看结局
4. 综合断吉凶
5. 落到用户具体问题
6. 给行动建议
```

---

## 七、举例：风山渐 第三爻动 → 风地观

- **本卦 风山渐**（上巽下艮）
  - 上卦巽为体（动爻在第三爻 = 下卦）
  - 下卦艮为用
  - 巽（木）vs 艮（土）→ 巽木克艮土 → 体克用 → **中吉，我能制对方**
- **互卦**：本卦 2-3-4 爻 = ?；3-4-5 爻 = ?（具体取决于本卦 binary）
- **变卦 风地观**（上巽下坤）
  - 巽（木）vs 坤（土）→ 仍是 体克用，但更大维度的「观望」格局
- **综合**：体克用 → 我能掌控；变卦观 → 提示要瞻望、看清；半仙断「续可以续，但**心**不能续——边干边看外面」

---

## 八、半仙引用此规则的方式

得到本卦 / 之卦 / 动爻后，半仙不要急着翻 64 卦解释，先按以下步骤：

1. 标出体用：「上 X 卦为体，下 Y 卦为用」（或反过来）
2. 看体用五行：「X 五行 vs Y 五行 = 生 / 克 / 比和」
3. 给出大方向：「大吉 / 中吉 / 平 / 小凶 / 大凶」
4. 然后再用 64 卦的卦辞 / 爻辞 解读细节
