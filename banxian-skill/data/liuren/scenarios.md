# 现代场景模板

源自 panpanmao `domains/zhanbu/liuren/knowledge/scenarios.ts` + `categories.ts`。

按问题类型分类，给出 6 神在该场景下的针对性解读模板。

---

## 一、感情类

| 神 | 解读模板 |
|---|---|
| **大安** | 关系稳定中。男女问感情：缘分尚浅但平和。需主动注入新意，避免日久生倦。 |
| **留连** | 感情拖泥带水、藕断丝连。**暗恋未表白者**：对方有意但未明朗。**已表白者**：对方反复犹豫。 |
| **速喜** | 朱雀主消息——**主动出击为上**。3-7 日内必有进展。但勿反复追问对方，避免压力过大。 |
| **赤口** | 警告：近期有口舌、争执之兆。**已在恋者**：避免直接对抗。**追求中者**：暂缓表白，等情绪平稳。 |
| **小吉** | 缘分到位，鸳鸯成对之象。**单身者**：近期有合适人出现。**已有对象**：可考虑下一步。 |
| **空亡** | 此事根基不实——可能对方无心，可能时机错位。**强求易伤**。 |

---

## 二、事业类

| 神 | 解读模板 |
|---|---|
| **大安** | 工作稳定但难有突破。**问跳槽**：现单位是「稳」，外面是「未知」，宜守不宜动。**问晋升**：积累尚不足。 |
| **留连** | 工作进展缓慢。**问项目**：会一拖再拖。**问跳槽**：会面试很多家但难下决定。 |
| **速喜** | 贵人星动！**问机会**：3 日内有消息。**问 offer**：会快速到来。**问签约**：宜速决。 |
| **赤口** | 职场有人事冲突——可能是同事、可能是上下级。**慎言慎行**，少在群里说话。 |
| **小吉** | 合作运强。**问合伙**：可成。**问联名/跨部门**：天作之合。 |
| **空亡** | 此事概念虚浮——**画饼为多**。问 offer 注意是否水分多，问项目注意是否真落地。 |

---

## 三、财运类

| 神 | 解读模板 |
|---|---|
| **大安** | 财运平稳，正财为主。**正财**：收入稳定。**偏财**：保守为主，不宜大动。 |
| **留连** | 财物会拖延。**问回款**：会慢但能收。**问投资**：流动性差，慎入。 |
| **速喜** | 偏财动象——3-7 日有进帐。**正财**：奖金、提成机会。**偏财**：小额收益。**不指具体数字！** |
| **赤口** | 财物上有「口」，意为破财、冲突。**慎签合同**，**避免合伙纠纷**。 |
| **小吉** | 小额可入但大额慎重。「小」字是关键——求小成大不可。 |
| **空亡** | 财来财去，留不住。**问投资**：恐失。**问理财**：先稳现金。 |

## 四、出行类

| 神 | 解读模板 |
|---|---|
| **大安** | 行人未动——若问家人朋友是否在路上，答案是「还没出发」。 |
| **留连** | 路上有阻——天气、堵车、临时变故。早出门为宜。 |
| **速喜** | 顺行——可去，行程顺利。 |
| **赤口** | 行途有口——避免与人冲突，尤其拼车、合住。 |
| **小吉** | 同行有缘——会遇到合得来的人。 |
| **空亡** | 行而无果——可能临时取消，或到了发现意义不大。 |

---

## 五、失物类

| 神 | 解读模板 |
|---|---|
| **大安** | 失物未远——**在家中近处**。东方位（青龙），木属性（纸张、家具下）。 |
| **留连** | 失物在「藏」中——衣物口袋、抽屉深处、不常用包。 |
| **速喜** | 失物在「光」处——南方位，光照能及（窗边、阳台、桌面）。 |
| **赤口** | 失物可能被「口」碰过——口袋、嘴边（餐桌、吃饭处）、或被他人借走未还。 |
| **小吉** | 失物有「合」——和别的东西在一起（成对、成套的物件附近）。 |
| **空亡** | 失物可能丢了——但也可能在你完全没想到的地方（"灯下黑"）。 |

---

## 六、健康类（合规：只看运势，不诊断）

| 神 | 解读模板 |
|---|---|
| **大安** | 整体康健，无大碍。仍需注意作息规律。 |
| **留连** | 慢性、反复——以情绪、消化、肝胆为主。**建议就医检查**。 |
| **速喜** | 急、热、上焦症状——容易上火、咽喉不适。**多饮水降火**。 |
| **赤口** | 注意口腔、呼吸道、刀伤、烫伤——**留心安全细节**。 |
| **小吉** | 整体平和，宜调养。 |
| **空亡** | 有「虚」证——疲劳、气虚、低血压等。**需补养**。 |

**合规底线**：半仙不诊断、不开药、不判生死，所有"建议就医"必加。

## 七、考试 / 学业

| 神 | 解读模板 |
|---|---|
| **大安** | 平稳发挥——按平时水平。 |
| **留连** | 易拖延、易紧张——考前需放松。 |
| **速喜** | 文思敏捷——发挥能超水平。 |
| **赤口** | 慎重审题——容易因看错题失分。 |
| **小吉** | 有贵人助——可能题目正合所学。 |
| **空亡** | 准备不足显现——临场需冷静。 |

## 半仙引用此章节的方式

得到主卦（如「速喜」）+ 用户问题分类（如「财运」）后，半仙应：
1. 找到对应单元格的模板（如 速喜 × 财运）
2. **结合用户具体场景** 落地（如"这个 offer 是大厂还是创业公司"）
3. 用半仙 voice 转述（不是直接念模板，而是用玄学语气重写）
