# 五行生克 + 6 神映射

## 五行相生（顺时针）

```
木 → 火 → 土 → 金 → 水 → 木 (循环)
```

| 生 | 含义 |
|---|---|
| 木生火 | 木燃则火生 |
| 火生土 | 火燃成灰为土 |
| 土生金 | 土中藏金 |
| 金生水 | 金属凝露为水 |
| 水生木 | 水滋养木 |

## 五行相克（隔位相克）

```
木 → 土 → 水 → 火 → 金 → 木 (循环)
```

| 克 | 含义 |
|---|---|
| 木克土 | 木之根破土 |
| 土克水 | 土能围堵水 |
| 水克火 | 水能灭火 |
| 火克金 | 火能融金 |
| 金克木 | 金器伐木 |

## 五行运用原则

1. **同我者比和**——卦中之神与日时五行同 → 加分
2. **生我者扶助**——卦中之神被日时五行所生 → 加分
3. **克我者削弱**——卦中之神被日时五行所克 → 减分
4. **我克者耗损**——卦中之神克日时五行 → 中性
5. **我生者泄气**——卦中之神生日时五行 → 略减分

## 应用：判断卦象旺衰

例：**速喜（火）落在午时（火）→ 比和加分** = 此卦特别有力，应事极快
例：**留连（水）落在午时（火）→ 我克 = 中性** = 留连本就拖延，午时（火）使其不再泄气，但本质仍是拖
