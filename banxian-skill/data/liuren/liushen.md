## 大安

- **别名**：青龙
- **性质**：大吉（评分 95/100）
- **五行**：木
- **方位**：东方
- **颜色**：青色、绿色
- **吉数**：1, 5, 7
- **吉时**：卯时(5-7点)、辰时(7-9点)
- **状态**：身不动时

### 经典断辞

```
大安：身未动时，属木青龙，谋事一五七。
断曰：大安事事昌，求财在坤方，失物去不远，宅舍保安康。
行人身未动，病者主无妨，将军回田野，仔细更推详。
```

### 核心语义

「身不动时」——静止、安稳、吉祥、平安

### 关键判断点

- 宜守不宜动，事在酝酿未发阶段
- 静待时机为上策，稳中有成
- 女问大吉，男问略平需保持新鲜
- 问健康无恙，问家宅平安
- 问行人未动身，问失物在近处

## 留连

- **别名**：玄武
- **性质**：小凶（评分 35/100）
- **五行**：水
- **方位**：北方
- **颜色**：黑色、深蓝
- **吉数**：2, 8, 10
- **吉时**：子时(23-1点)、亥时(21-23点)
- **状态**：卒未归时

### 经典断辞

```
留连：卒未归时，属水玄武，谋事二八十。
断曰：留连事难成，求谋日未明，官事只宜缓，去者未回程。
失物南方见，急讨方趁心，更须防口舌，人口且平平。
```

### 核心语义

「人未归时」——拖延、纠缠、暧昧不明、犹豫反复

### 关键判断点

- 时机未到，事情难以速成
- 需反复多次方能成功
- 感情代表欲罢不能、藕断丝连
- 问行人久久未归
- 问事业进展缓慢，耐心等待

### 正向转化 / 化解之道

留连落在此位，暗示这件事需要更多时间来酝酿。这不是坏事——很多好结果都需要足够的铺垫。

建议：
• 利用等待期做更充分的准备
• 不要急于推进，欲速则不达
• 关注"还没准备好"的环节

记住：留连不是"不行"，而是"再等等"。

## 速喜

- **别名**：朱雀
- **性质**：中吉（评分 85/100）
- **五行**：火
- **方位**：南方
- **颜色**：红色、紫色
- **吉数**：3, 6, 9
- **吉时**：午时(11-13点)、未时(13-15点)
- **状态**：人便至时

### 经典断辞

```
速喜：人便至时，属火朱雀，谋事三六九。
断曰：速喜喜来临，求财向南行，失物申未午，逢人路上寻。
官事有福德，病者无祸侵，田宅六畜吉，行人有信音。
```

### 核心语义

「人便至时」——快速、喜庆、时机已到

### 关键判断点

- 喜讯将至，宜立即行动
- 拖延反而不利，速决速行
- 问感情发展迅速，一见倾心
- 问事业有好消息
- 问行人即将归来
- 火性急，主快不主久

## 赤口

- **别名**：白虎
- **性质**：中凶（评分 30/100）
- **五行**：金
- **方位**：西方
- **颜色**：白色、银色
- **吉数**：4, 7, 10
- **吉时**：酉时(17-19点)、申时(15-17点)
- **状态**：官事凶时

### 经典断辞

```
赤口：官事凶时，属金白虎，谋事四七十。
断曰：赤口主口舌，官非切要防，失物急去寻，行人有惊慌。
六畜多作怪，病者出西方，更须防诅咒，恐怕染瘟殃。
```

### 核心语义

「官事凶时」——口舌是非、争执、凶险

### 关键判断点

- 易生争吵、诉讼，需防小人
- 言多必失，祸从口出
- 问感情多争吵、分手之象
- 问事业需防官非
- 问健康注意口腔、呼吸道
- 宜静不宜动，少说多做

### 正向转化 / 化解之道

赤口提醒：近期挑战可能来自沟通层面。成败关键在于"怎么说"和"对谁说"。

建议：
• 重要沟通前先打草稿
• 涉及利益的事留书面记录
• 少参与是非讨论

白虎虽凶，却也主行动力和决断。用好这股能量，它就是助力。

## 小吉

- **别名**：六合
- **性质**：小吉（评分 75/100）
- **五行**：木
- **方位**：西南（坤方）
- **颜色**：绿色、青色
- **吉数**：1, 5, 7
- **吉时**：辰时(7-9点)、巳时(9-11点)
- **状态**：人来喜时

### 经典断辞

```
小吉：人来喜时，属木六合，谋事一五七。
断曰：小吉最吉昌，路上好商量，阴人来报喜，失物在坤方。
行人即便至，交易甚是强，凡事皆和合，病者叩穹苍。
```

### 核心语义

「人来喜时」——和合、顺遂、贵人相助

### 关键判断点

- 诸事和合，有人牵线
- 问感情有人介绍，姻缘可成
- 问合作双方能谈拢
- 女贵人相助更明显
- 只主小吉不主大吉，稳中有进

## 空亡

- **别名**：勾陈
- **性质**：大凶（评分 15/100）
- **五行**：土
- **方位**：中央
- **颜色**：黄色、褐色
- **吉数**：5, 10, 12
- **吉时**：丑时(1-3点)、未时(13-15点)
- **状态**：音信稀时

### 经典断辞

```
空亡：音信稀时，属土勾陈，谋事三六九。
断曰：空亡事不祥，阴人多乖张，求财无利益，行人有灾殃。
失物寻不见，官事有刑伤，病人逢暗鬼，祈解保安康。
```

### 核心语义

「音信稀时」——落空、无果、忧虑

### 关键判断点

- 事情难成，努力可能付诸东流
- 问失物永难寻回
- 问感情缘尽宜放
- 但「空亡主散」——问破财、消灾反应验
- 此时不宜妄动，韬光养晦

### 正向转化 / 化解之道

空亡落此位，这件事按目前方向恐难如愿。但请理解：空亡不是惩罚，是保护——帮您及早看清这条路走不通。

"此路不通"的背后，往往藏着"另有安排"。

建议：
• 重新评估这件事是否是真正想要的
• 考虑其他方式达成目标
• 调整策略后再尝试

记住：结束也是一种开始。
