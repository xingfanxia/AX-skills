# 三宫吉凶组合断辞

源自 panpanmao `domains/zhanbu/liuren/knowledge/combinations.ts`。

## 三宫角色

- **月宫** = 天时 / 起因 / 远因
- **日宫** = 地利 / 过程 / 当下
- **时宫** = 人和 / 结果 / 主卦

吉凶判定：大安/速喜/小吉 = 吉，留连/赤口/空亡 = 凶。

## 8 种组合（吉凶 pattern）

### 吉吉吉 — 大顺
> 起因吉、过程吉、结果吉。三阶皆顺，事可成且无后患。
> 半仙：「卦中三宫皆吉——此事可放胆为之。」

### 凶吉吉 — 先难后易
> 起因不利，但中段地利在我，结果亦吉。
> 半仙：「开局虽有阻，然过程渐顺，终得善果。古人云『万事开头难』，正是此卦。」

### 吉凶吉 — 中段波折
> 起因吉、过程有阻、结果终吉。
> 半仙：「此事看似顺起，中段有变。然天意眷顾，结果仍吉。需注意中途之变。」

### 吉吉凶 — 警示
> 起因吉、过程顺，但结果不利。
> 半仙：「此卦最当警惕——前段顺遂易让人放松，结果反成败笔。须知『行百里者半九十』。」

### 凶凶吉 — 苦尽甘来
> 起因凶、过程艰，但结果反吉。
> 半仙：「此卦虽前段坎坷，然否极泰来。若能挺过中段，必有甘果。」

### 凶吉凶 — 短暂回光
> 起因凶、中段一时之顺，结果仍凶。
> 半仙：「此卦中段易令人误判，以为事可为之。然始终凶，是表象之顺，不可信。」

### 吉凶凶 — 由顺转衰
> 起因吉、过程不利、结果亦凶。
> 半仙：「此卦最忌恋旧——前段顺遂时不思变，等到中段变化已不可挽。须及早调整。」

### 凶凶凶 — 大不利
> 三阶皆凶，此事不宜。
> 半仙：「卦中三宫皆凶，此事当退不当进。强为之必有损。」

## 三同特殊（Triple Same）

当月宫 = 日宫 = 时宫（同一神位 3 次出现）：

| 同 | 半仙断 |
|---|---|
| 三大安 | 「极静极安——若问行人则未动，若问求事则未发，需主动出击。」 |
| 三留连 | 「极拖极缠——此事陷在原地，需打破僵局。」 |
| 三速喜 | 「极速极喜——此事将在 3 日内见分晓。」 |
| 三赤口 | 「极冲极口舌——务必慎言慎行，避免争执。」 |
| 三小吉 | 「极合极利——天作之合之象，把握当下。」 |
| 三空亡 | 「极空极虚——此事根基不稳，宜暂缓。」 |

## 半仙引用此章节的方式

得到三宫 pattern 后（如 `吉凶吉`），半仙应：
1. 描述 pattern 的整体走向
2. 落到用户具体问题上
3. 给行动建议（动 / 守 / 变 / 等）
