# 世应 + 用神选择

> 来源：panpanmao `domains/zhanbu/liuyao/data/palace.ts` 的 `SHIYAO_POSITION` 表。

## 一、世爻 vs 应爻

| 角色 | 含义 |
|---|---|
| **世爻（世）** | 自己（求测人 / 主体） |
| **应爻（应）** | 对方（事 / 人 / 对手 / 客体） |

**世应永远相距 3 位**——这是规则。

---

## 二、世应位置（按八宫卦序）

每卦的世应位置由**它在卦宫中的位置**决定。卦宫共 8 个，每宫 8 卦，按变爻顺序排：

| 宫内位置 | 卦类 | 世爻位 | 应爻位 |
|---|---|---|---|
| 1 | **本宫卦**（八纯卦） | 上爻（6） | 三爻（3） |
| 2 | 一变卦 | 初爻（1） | 四爻（4） |
| 3 | 二变卦 | 二爻（2） | 五爻（5） |
| 4 | 三变卦 | 三爻（3） | 上爻（6） |
| 5 | 四变卦 | 四爻（4） | 初爻（1） |
| 6 | 五变卦 | 五爻（5） | 二爻（2） |
| 7 | **游魂卦** | 四爻（4） | 初爻（1） |
| 8 | **归魂卦** | 三爻（3） | 上爻（6） |

例：
- 乾为天 = 乾宫第 1 卦（本宫卦） → 世在六爻，应在三爻
- 天风姤 = 乾宫第 2 卦（一变） → 世在初爻，应在四爻
- 火地晋 = 乾宫第 7 卦（游魂） → 世在四爻，应在初爻
- 火天大有 = 乾宫第 8 卦（归魂） → 世在三爻，应在上爻

## 三、世应关系断吉凶

| 关系 | 解读 |
|---|---|
| 世应相生 | 我与对方 / 事和谐，事易成 |
| 世应比和 | 平稳，无大波 |
| 世应相克：世克应 | 我占优势 |
| 世应相克：应克世 | 我被动，对方占优 |
| 世空 | 我犹豫、心不诚、力不足 |
| 应空 | 对方无心、事难成 |
| 世应皆空 | 此事虚耗，难以成 |

---

## 四、世应 × 用神位置

| 用神位置 | 含义 |
|---|---|
| 用神在世爻 | 此事关乎自己 |
| 用神在应爻 | 此事关乎对方 |
| 用神在间爻（世应之间） | 中间人 / 媒介有作用 |
| 用神在世应外（不在世应路径上） | 边缘因素 |

## 五、看局原则（半仙断卦的整体框架）

```
1. 识用神（参考 liuqin.md）
2. 看用神是否旺相（参考月建/日建的支持）
3. 看用神有无动爻（动则有变）
4. 看用神位置 vs 世应（在世说自己事，在应说对方事）
5. 看世应关系（自己 vs 对方的态势）
6. 看变卦（最终走向）
7. 看忌神 / 仇神（外部干扰因素）
8. 综合断：吉吉 / 吉中藏凶 / 凶中藏吉 / 凶凶
```

## 六、半仙引用顺序

得到本卦 + 之卦 + 动爻位 + 用神 后，半仙应：

1. 「世爻在第 X 爻，应爻在第 Y 爻——你和对方的关系是 [生/克/比]。」
2. 「用神（如妻财/官鬼）落在第 Z 爻，是 [世/应/间] 位——主 [自己事/对方事/中间人]。」
3. 「用神 [旺/衰/空/动] ——主 [有力/无力/落空/有变]。」
4. 然后落到用户具体问题，给行动建议。

## 七、特殊情况

- **用神不在卦中**：6 爻中没有用神所属六亲 → 「飞神伏神」断法（panpanmao 暂未实现，此 skill v1 不处理）
- **多个用神**：6 爻中有多个用神（如多个妻财） → 取动爻者优先；若都静则取世应位上的
- **用神持世**：用神就是世爻 → 关乎自己根本，主动权在我
