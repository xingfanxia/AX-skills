# 六亲断法

六爻断卦的核心——**六亲**（父母 / 官鬼 / 兄弟 / 妻财 / 子孙）。

> 来源：panpanmao `domains/zhanbu/liuyao/data/palace.ts` 的 `getLiuQin` 规则。

---

## 一、六亲含义

| 六亲 | 与卦宫五行关系 | 类比 | 主断 |
|---|---|---|---|
| **父母** | 生卦宫者 | 父辈、师长、单位、文书、衣服、车 | 学业、文凭、合同、长辈、保护伞 |
| **官鬼** | 克卦宫者 | 官位、丈夫（女问）、敌人、病灾 | 工作、官司、灾祸（女问婚姻） |
| **兄弟** | 比和（同五行） | 兄弟、朋友、同行、对手 | 合作、争夺、破财、群体 |
| **妻财** | 卦宫所克者 | 妻子（男问）、财物、仆役 | 财运、女性、奴仆（男问婚姻） |
| **子孙** | 卦宫所生者 | 子女、晚辈、宠物、医药 | 子女、解忧、解药、福德 |

## 二、六亲推算公式

以**卦宫五行**为「我」（gongWuXing），6 爻地支五行为「彼」（yaoWuXing）：

| gongWuXing 与 yaoWuXing 关系 | 六亲 |
|---|---|
| yaoWuXing 生 gongWuXing | 父母 |
| yaoWuXing 克 gongWuXing | 官鬼 |
| 比和（同） | 兄弟 |
| gongWuXing 克 yaoWuXing | 妻财 |
| gongWuXing 生 yaoWuXing | 子孙 |

例：**乾宫卦（金）**，6 爻地支五行分别推：

| 爻位地支五行 | 与金的关系 | 六亲 |
|---|---|---|
| 金 | 比和 | 兄弟 |
| 木 | 金克木 | 妻财 |
| 水 | 金生水 | 子孙 |
| 火 | 火克金 | 官鬼 |
| 土 | 土生金 | 父母 |

## 三、用神选择（按问题类型）

不同问题看不同六亲（用神）：

| 问题类型 | 用神 | 备注 |
|---|---|---|
| 婚姻（男问） | 妻财 | 男以财为妻 |
| 婚姻（女问） | 官鬼 | 女以官为夫 |
| 财运 | 妻财 | 主财物 |
| 工作 / 升迁 | 官鬼 | 官位 |
| 官司 / 是非 | 官鬼 | 注意官鬼旺则不利 |
| 学业 / 考试 | 父母 | 文凭文书 |
| 合同 / 文书 | 父母 | 任何文书凭证 |
| 长辈 / 师长 | 父母 | |
| 子女 | 子孙 | |
| 医药 / 解忧 | 子孙 | 子孙能解官鬼 |
| 兄弟朋友 | 兄弟 | |
| 合伙合作 | 兄弟 | 兄弟旺则有竞争 |

## 四、用神状态判断

判断用神「旺 / 衰 / 空 / 动」：

| 状态 | 判断标准 | 含义 |
|---|---|---|
| **旺相** | 月建生扶 / 日建生扶 | 吉，有力 |
| **休囚** | 月建克泄 / 日建克泄 | 凶，无力 |
| **空亡** | 在旬空范围内 | 落空，等出空时应事 |
| **动爻** | 摇钱时为 6 或 9（老阴/老阳） | 主事，动则有变 |
| **暗动** | 静爻被日辰所冲 | 看似不动实则动 |

## 五、忌神 + 仇神

除了用神，还要看：

- **忌神** = 克用神的爻 → 忌神旺则用神受损
- **仇神** = 生忌神的爻 → 仇神旺则忌神得力

例：男问婚姻，妻财为用神，兄弟为忌神（兄弟克妻财），父母为仇神（父母生兄弟）。
若兄弟动而旺 → 婚姻有阻（朋友干涉 / 同性竞争）。

## 六、半仙断卦时的引用顺序

```
1. 先识用神（按问题类型对照表）
2. 看用神所在爻位（初/二/三/四/五/上）
3. 看用神是否在世应位（在世说自己事，在应说对方事）
4. 看用神旺衰（月日扶/克）
5. 看用神有无空亡 / 动爻
6. 看忌神状态（克用神的爻是否旺动）
7. 综合断
```
